# Security Policy

## Supported Versions

The current `main` branch is supported.

## Reporting A Vulnerability

Please report vulnerabilities through GitHub Security Advisories when available, or by opening a private issue with the repository owner.

Do not include secrets, private screenshots, customer data, or production credentials in public issues.

## Scope

This project automates local browser audits and static file serving. Security-sensitive areas include:

- local file serving behavior
- environment variable handling
- browser automation against authenticated applications
- accidental capture of private data in screenshots

Consumers should review screenshots before sharing them externally.
